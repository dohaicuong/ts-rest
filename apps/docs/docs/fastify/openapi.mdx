# OpenAPI

Serve the document generated by `@ts-rest/open-api` with `@fastify/swagger` and @fastify/swagger-ui

## Installation

<InstallTabs packageName="@fastify/swagger @fastify/swagger-ui" />

## Generating a OpenAPI Document

```typescript
import { myContract } from './my-api';
import { generateOpenApi } from '@ts-rest/open-api';

const openApiDocument = generateOpenApi(myContract, {
  info: {
    title: 'Posts API',
    version: '1.0.0',
  },
});
```

## Register fastify plugin

```typescript
import fastify from 'fastify'
import fastifySwagger from '@fastify/swagger'
import fastifySwaggerUI from '@fastify/swagger-ui'

import { openApiDocument } from './document'

const app = fastify()
  .register(fastifySwagger, {
    swagger: openApiDocument
  })
  .register(fastifySwaggerUI)
```
